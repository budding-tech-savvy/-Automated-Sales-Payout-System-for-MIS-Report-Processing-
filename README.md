*Project Overview*

Built an AI-powered analytics assistant using Databricks Genie that enables business users to query transportation and logistics data using natural language instead of writing SQL or navigating multiple dashboards.

*Problem Statement*

Business users depended on data analysts for transportation-related reports, resulting in long turnaround times and increased manual effort.

*Solution*
Integrated transportation datasets into Databricks.
Created AI-ready semantic data resources.
Configured Databricks Genie with business-specific instructions.
Enabled employees to ask questions in natural language.
Returned SQL-generated insights automatically.

*Technologies*
Databricks Genie
SQL
Databricks SQL Warehouse
Delta Tables

*Features*

Natural language analytics
Transportation KPI analysis
Shipment tracking
Carrier performance analysis
Order analytics
Interactive conversational interface
Business Impact
Reduced report turnaround from days/weeks to minutes.
Reduced dependency on analysts for routine reporting.
Improved accessibility of transportation insights.
Accelerated operational decision-making.
